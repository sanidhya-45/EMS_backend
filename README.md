# EMS_backend

This repository contains the backend of Employee Management System (EMS), it is built using Java and Spring Boot and Postgres.
<br>
The following project demonstrates the employee management system which contains features like:
<br>
  - Add an Employee
  - Update an Employee details
  - Get all employees
  - Delete all employees
  - Get employee by Id
  - Delete employee by Id
<br>
The project uses REST api services provided by Spring controller, the local tests were done using Postman.
